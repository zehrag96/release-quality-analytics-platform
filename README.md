# Release Quality Analytics Platform

## Project Overview
This project simulates an enterprise-level QA analytics platform that helps monitor software release quality.
It provides insights into releases, test execution, bug tracking and quality metrics using SQL, Python and Power BI.

## Technologies
- SQLite (Database)
- SQL (Data Analysis)
- Python (Data Processing)
- Power BI (Dashboard & Visualization)

## Project Goals
- Build a relational database for release management.
- Analyze software quality metrics using SQL.
- Generate reports with Python.
- Create an interactive dashboard with Power BI.
- Simulate a real-world enterprise QA reporting solution.
